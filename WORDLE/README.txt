Java Console Wordle Game
Introduction:
This Java console application is a simple implementation of the popular word-guessing game, Wordle. Players must guess a hidden word within a limited number of attempts.

How to Play:
Starting the Game:
    Run the Java application on your console or terminal.
    The program will generate a random (5-to-8)-letter word that you need to guess.
Making a Guess:
    Enter between a five to eight lettered word as your guess
The game will provide feedback on your guess:
    Correct letters in the correct position will be marked with green.
    Correct letters in the wrong position will be marked with yellow.
    Incorrect letters will be marked with red.
Guessing Strategy:
    Use the feedback provided to refine your guesses.
    Guess the word within the limited number of attempts to win the game.
End of Game:
    You win if you correctly guess the word within the specified attempts.
    You lose if you exhaust all attempts without guessing the word.

Requirements:
    Java Development Kit (JDK) installed on your system.
How to Run:
    Compile the source code using the Java compiler:
        Copy code
        javac Wordle.java
    Run the compiled Java program:
        Copy code
        java Wordle
Additional Notes:
    This game is designed to run in a console or terminal environment.
    The wordlist used for generating random words can be customized or extended as needed.
    Feel free to modify the code to enhance or customize the gameplay experience.
